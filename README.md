# Relatório de Missões: Automação e IA (ADS)

Este repositório contém o desenvolvimento das Missões 01 a 03 da disciplina, utilizando ferramentas de Inteligência Artificial para auxílio na codificação e refatoração.

##  Estrutura do Projeto
- `missao1_organizador.py`: Script de automação para organização de ficheiros.
- `missao2_viacep.py`: Integração com API ViaCEP e tratamento de erros.
- `missao3_sensor.py`: Sistema modular de monitorização desenvolvido com Classes.

##  Uso de IA e Reflexão Crítica
Conforme solicitado nas metas de sucesso, o desenvolvimento seguiu o modelo de colaboração humano-IA:

### Missão 01 e 02 (Replit Agent)
Utilizei o Replit Agent para a lógica inicial. Na Missão 02, provoquei um erro intencional na URL para testar a capacidade de correção da IA. O resultado foi uma implementação robusta de `try/except`, garantindo que o programa não falhe em caso de erros de rede.

### Missão 03 (GitHub Copilot)
Nesta missão, utilizei o GitHub Copilot para criar uma estrutura de Programação Orientada a Objetos (POO). O Copilot sugeriu a criação das classes `TemperatureSensor` e `Logger`, o que tornou o código modular e fácil de manter.

##  Conclusão
A integração de diferentes ferramentas (Replit e Copilot) demonstrou como o uso de IAs pode acelerar o "Pipeline" de desenvolvimento, permitindo focar na arquitetura e na qualidade da entrega final.
